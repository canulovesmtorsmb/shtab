# Голоса для ассистента ШТАБ — большая подборка вариантов

Дата: 22 июня 2026. Цель: заменить «так себе» встроенный браузерный голос на естественный, чтобы звучало хорошо и на iPhone, и на компьютере.

## Как это будет работать (архитектура)

Сейчас озвучка идёт через встроенный голос браузера (Web Speech API) — он зависит от системы и часто звучит роботично, особенно по-русски.
Улучшенный путь: добавить в твой Worker эндпоинт /tts. Сайт отправляет текст в Worker, Worker зовёт облачный TTS (ключ хранится секретом на сервере), получает аудио mp3 и отдаёт сайту, сайт проигрывает обычным аудио-плеером. Работает одинаково на айфоне и на компьютере, ключ в браузере не светится, CSP менять не нужно (домен Worker уже разрешён).

Open-source модели (Silero, Piper, Coqui) в этот путь не вписываются: их нельзя запустить ни в браузере, ни в Cloudflare Worker — нужен отдельный сервер (VPS с Python). Это отдельная история, отметил ниже честно.

## A. Облачные TTS — подключаются к Worker /tts (рекомендуемый путь)

| Провайдер | Русский голос | Бесплатно | Подключение | Заметки |
|-----------|---------------|-----------|-------------|---------|
| Microsoft Azure Neural | Очень хороший, родные ru-RU (Dmitry, Svetlana, Dariya и др.) | ~500 000 символов/мес | средне (ключ + регион) | Лучший баланс качество/бесплатно. Много голосов и стилей. |
| Google Cloud TTS | Хороший, ru-RU WaveNet/Neural2 | ~1 млн символов/мес (WaveNet) | средне (ключ/сервис-аккаунт) | Очень щедрый бесплатный лимит. |
| Yandex SpeakKit v3 | Самый натуральный русский (родной) | пробный период/гранты, далее платно | просто (ключ) | Лучшее звучание по-русски. Рублёвая оплата. Цены уточняй, меняются. |
| OpenAI TTS (tts-1 / gpt-4o-mini-tts) | Нормальный, голоса не родные русские | платно, дёшево | очень просто (один ключ, как у тебя /ask) | Проще всего подключить, единый стиль с мозгом. |
| ElevenLabs | Топ по эмоциям, но русский не родной | ~10 000 символов/мес | просто (ключ) | Дорого для постоянного, free очень маленький. |

Мой выбор по балансу: Azure Neural (бесплатно и качественно по-русски) или OpenAI TTS (проще всего, раз у тебя уже Groq-ключ в Worker). Самый красивый русский — Yandex SpeakKit, но платный.

## B. Бесплатный «хак» — голоса Microsoft Edge без ключа

- openai-edge-tts (travisvn): https://github.com/travisvn/openai-edge-tts — бесплатный self-hosted сервер, отдаёт голоса Microsoft Edge (те же нейронные ru-RU, что и Azure), API в формате OpenAI. Нужен маленький сервер (или запуск локально). Качество как у Azure, без ключа и лимитов, но требует хостинга.

## C. Open-source с GitHub (нужен свой сервер, не для статичного сайта)

| Проект | Что это | Русский | Ссылка |
|--------|---------|---------|--------|
| Silero models | Лёгкие нейро-TTS, голоса aidar, baya, kseniya, xenia, eugene, авто-ударения | Отлично, родной | https://github.com/snakers4/silero-models |
| Piper (piper1-gpl) | Быстрый офлайн VITS, работает даже на Raspberry Pi, естественное звучание | Есть ru-голоса | https://github.com/OHF-Voice/piper1-gpl |
| Coqui TTS (XTTS v2) | Клонирование голоса по короткому образцу, многоязычный | Поддержка ru | https://github.com/coqui-ai/TTS |
| openedai-speech | Self-host сервер (Piper/XTTS) с API в формате OpenAI — удобная обёртка | через Piper/XTTS | https://github.com/matatonic/openedai-speech |

Плюс: бесплатно, приватно (данные не уходят в облако), Silero даёт очень приличный русский. Минус: нужен сервер с Python (мини-VPS), Cloudflare Worker их не запустит. Для тебя это означает дополнительную инфраструктуру.

## Рекомендация под твой случай

1. Хочешь «просто и хорошо, на всех устройствах, бесплатно» → Azure Neural через Worker /tts. Качественный русский, щедрый бесплатный лимит.
2. Хочешь «минимум возни» (у тебя уже есть Worker и ключи) → OpenAI TTS через /tts: один ключ, пара строк в Worker.
3. Хочешь «самый красивый русский» и готов платить → Yandex SpeakKit.
4. Хочешь «бесплатно и приватно, без облака» и не против поднять сервер → Silero или openai-edge-tts на мини-VPS.

## Что дальше

Скажи, какой вариант берём — я добавлю в Worker эндпоинт /tts под выбранный провайдер и переключу ассистента с браузерного голоса на качественное аудио (с запасным вариантом на встроенный голос, если сеть недоступна). От тебя понадобится только ключ провайдера (задаётся секретом в Worker, как AI_API_KEY).
